This week's material included `iterators` and `generators`.

_Universal_
Iterators are objects which provide the functionality for traversing a data structure.

_Python specific_
Generators are the way which `python` uses to implement iterators most efficiently. It provides a common way for utilizing this functionality through the `__iter__` and `__next__` methods. Implementing these methods gives the object ability to be "traversed".

**Generators**
They are memory and speed efficient and thus - very useful.
