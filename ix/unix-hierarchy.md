_/_ -> the root dir
_/bin_ -> essential low-level utilities
_/usr/bin_ -> higher-level utilities
_/sbin_ -> superuser system utilities
_/lib_ -> program libs
_/usr/lib_ -> program libs for higher-level programs
_/tmp_ -> temp file storage (avail. to any user)
_/home_ -> for each user...
_/etc_ -> configuration
_/dev_ -> hardware devices
_/proc_ -> pseudo-fs; used as an interface between the use and the kernel
